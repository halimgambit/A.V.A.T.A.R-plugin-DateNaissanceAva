# DateNaissanceAva
AVATAR donne son age depuis sa naissance

This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework.

🎯 Usage
Commandes :

tu as quel age, quel est ta date de naissance.

# Multi-room
The DateNaissanceAva plugin is fully multi-room.

# Multi-language
The DateNaissanceAva plugin relies solely on the system's available languages.
